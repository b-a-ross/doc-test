Welcome to PSPSPS documentation!
===================================

**PSPSPS** (/PSPSPS/) is the polished and sourced power system parameter suite 
that lets you quickly build reasonable, detailed power system models from basic equipment data. 

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development.
